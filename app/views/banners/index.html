Banners

How to give users feedback using banners.

We use a front-loaded pattern for the main message which tends to mean it becomes passive. This means we can be direct and economical with space. Being passive isn't a problem as long as it's obvious who's done the action.

We then include a drop lede to flesh out the implications of the action that has happened.

For example:

Invitation sent
You've sent an invitation to pete.kowalczyk@digi2al.co.uk.

Agreement signed
Good Content ltd can now spend levy funds.

100/UX00019 has been added
Levy funds from this PAYE scheme will now credit your account.

Invitation cancelled
You've cancelled the invitation sent to pete.kowalczyk@digi2al.co.uk.
