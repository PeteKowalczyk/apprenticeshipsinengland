CLART

How to write for the web, and write for users.

Our style is based on the GDS style guide.

We write in plain English so that we don’t exclude people based on their education, cognitive function or reading ability. And we write in an accessible and structured way.

Readability

Plain English

We use plain English because it’s inclusive and easier for everyone to understand. Research from Harvard University has found that people at all literacy levels prefer plain English.

Teaching patients with low literacy skills, by Doak, Doak, & Root, 1996, Harvard, School of public health, page 7.

F-shaped pattern

We put important information at the top of web pages and front-load each sentence with the most important details. This is because Nielsen Norman Group (2006) showed that users scan web pages in a F-shaped pattern, scanning down the left side of the screen and reading across when they find something interesting.

Nielsen, J. (2006) ‘F-shaped pattern for reading web content’ http://www.nngroup.com/articles /f-shaped-pattern-reading-webcontent/

Scannability

We make pages scannable by using short sections of text, headers, bulleted lists to help people find information quickly and help that perform their tasks. Morkes and Nielsen (1997) found that by doing this users were able to perform tasks faster with fewer errors and better information recall.

Nielsen’s general recommendations for scannable text:

  • highlighted keywords (for example, using bold type or colour)
  • subheads
  • bulleted lists
  • clear organisation of paragraphs
  • concise language

Morkes, J. and J. Nielsen, (1997) ‘Concise, SCANNABLE, and objective: how to write for the web’ http://www.nngroup.com/articles /concise-scannable-andobjective-how-to-write-for-theweb/ Nielsen, J. (1997b) ‘Be succinct! (Writing for the web)’ http://www.nngroup.com/articles /be-succinct-writing-for-the-web/

Additonal style considerations

Don’t use a hyphen unless it’s confusing without it

Don’t use e.g. and i.e. Use 'for example' or ‘like’

Try to start sentences with the user’s objective. For example: To spend funds, you need to sign an agreement for Tesco plc
