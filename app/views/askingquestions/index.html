Asking questions

How we create a clear dialogue with users.

We default to radio buttons when asking questions or offering options.

Try to use a Yes / No pattern as it tends to focus the question. For example, rather than:

Who'll sign agreements?
Me / Someone else

Try to reposition questions something like this:

Sign agreements now?
Yes / No, not now

Add more PAYE schemes now?
Yes / No, not now

Also, try to be as explicit as you can in the answers.

It’s best if the answers work as sentences in their own right, but use your judgement as to whether or not the answers create too much repetition.

A good example of helpful answers is:

Send change request now?
Yes, send change request to employer
No, don’t make any changes
