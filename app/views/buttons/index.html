Buttons

How to format text for buttons.

For start buttons we simply use the label Start.

For a green button at the start of a transaction the label needs to say what the purpose of the transaction is, eg Add new team member.

Often within a transaction the green button simply moves the user onto the next step, in which case the default label is Continue.

For the final step in a transaction, the current convention is to use Continue for the label and then give the user a confirmation message once they've completed the transaction. We currently don't use 'Finish' for the final buttons in a transaction as this can cause confusion if the user's session isn't actually finishing.

We don’t use articles in green buttons. As such, we don’t need them to be first or second person. Your / my etc. So:

Rather than: Find your training providers or Find my training providers
Say: Find training providers

Rather than: Filter your apprentices or Filter my apprentices
Say: Filter apprentices
